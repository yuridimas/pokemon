# LARAVEL POKE API

Hit poke api lalu disimpan kedalam mongodb terdapat rate limit 5x dalam 1 menit dan menggunakan bearer token

## Requirement

- php 8.1
- php mongo extension
- mongodb
- postgresql / mysql

## Install

``` php
composer install
```

``` text
cp .env.example .env
```

config your mongodb connection

```php
php artisan serve
```
